# 编解码模块

## 介绍
编解码模块主要用于对数据进行编码和解码，主要包括以下功能：

- 编码：将指令结合其协议格式转换为二进制数据。
- 解码：将二进制数据转换为指令。

## 编码
编码是指将指令结合其协议格式转换为二进制数据。

1. 指令格式：指令格式是指指令的结构、字段、数据类型等。
2. 协议格式：协议格式是指协议的结构、字段、数据类型等。
